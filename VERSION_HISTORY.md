# Version History

This document lists the production releases of MetaXuda and the main purpose of each version.

## MetaXuda 1.0

**Synchronous CUDA-to-Metal runtime**

The first production release for running Numba CUDA kernels on Apple Silicon through Metal.

## MetaXuda 2.0

**Asynchronous CUDA-to-Metal runtime**

Added asynchronous execution, streams, managed execution, tiered memory, pipelines, and fusion.

## MetaXuda 2.0.2

**Compatibility update**

Current production release.

Added Metal 3 support for macOS Sequoia, automatic Metal 3/Metal 4 fallback, and fixes for implicit NumPy host-array transfers.

## MetaXuda 3.0

**Experimental engine and XGBoost support**

Currently under development. This release will include the experimental engine, XGBoost support, and known bug fixes.
